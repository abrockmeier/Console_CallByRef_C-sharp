# Console_CallByRef_C-sharp

Zusammenfassend lässt sich feststellen, dass sich eine Wert- oder Referenzübergabe bei
Referenztypen nur dann auswirkt, wenn in der aufgerufenen Methode der Parameter durch
Zuweisung einer neuen Referenz überschrieben wird. Es gelten dabei dieselben Gesetze wie
bei den Wertetypen.
